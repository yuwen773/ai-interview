## 4. 关键技术实现

### 4.1 统一响应封装

```java
public class Result<T> {
    private final Integer code;
    private final String message;
    private final T data;

    public static <T> Result<T> success(T data) {
        return new Result<>(200, "success", data);
    }

    public static <T> Result<T> error(String message) {
        return new Result<>(500, message, null);
    }

    public boolean isSuccess() {
        return this.code == 200;
    }
}
```

### 4.2 全局异常处理

```java
@RestControllerAdvice
public class GlobalExceptionHandler {
    @ExceptionHandler(BusinessException.class)
    public Result<?> handleBusinessException(BusinessException e) {
        return Result.error(e.getCode(), e.getMessage());
    }

    @ExceptionHandler(Exception.class)
    public Result<?> handleException(Exception e) {
        log.error("系统异常", e);
        return Result.error(500, "系统异常，请稍后重试");
    }
}
```

### 4.3 限流实现

```java
@Aspect
@Component
public class RateLimitAspect {
    @Around("@annotation(rateLimit)")
    public Object around(ProceedingJoinPoint pjp, RateLimit rateLimit) {
        // 1. 构建 Redis Key
        // 2. 执行 Lua 脚本检查限流
        // 3. 超限则抛出异常
        // 4. 未超限则执行业务逻辑
    }
}
```

**Lua 脚本** (原子性保证):
```lua
local key = KEYS[1]
local limit = tonumber(ARGV[1])
local expire = tonumber(ARGV[2])

local current = redis.call('GET', key)
if current == false then
    redis.call('SET', key, 1)
    redis.call('EXPIRE', key, expire)
    return 1
else
    if tonumber(current) < limit then
        redis.call('INCR', key)
        return 1
    else
        return 0
    end
end
```

## 7. 安全机制

### 7.1 文件上传安全

```java
// 1. 文件类型白名单
Set<String> ALLOWED_TYPES = Set.of("application/pdf", "application/vnd.openxmlformats-officedocument.wordprocessingml.document");

// 2. 文件大小限制
resume.setMaxSize(10 * 1024 * 1024); // 10MB

// 3. 内容类型验证
if (!ALLOWED_TYPES.contains(file.getContentType())) {
    throw new BusinessException(ErrorCode.RESUME_FILE_TYPE_NOT_SUPPORTED);
}

// 4. 文件哈希去重
String fileHash = DigestUtils.sha256Hex(file.getInputStream());
if (resumeRepository.existsByFileHash(fileHash)) {
    throw new BusinessException(ErrorCode.RESUME_DUPLICATE);
}
```

### 7.2 SQL 注入防护

- 使用 JPA 参数化查询
- 禁止动态 SQL 拼接
- 输入参数验证

### 7.3 XSS 防护

- 前端输入转义
- 后端响应头设置 `X-Content-Type-Options: nosniff`
