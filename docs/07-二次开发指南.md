# 二次开发指南

## 1. 扩展点概览

AI-Interview 项目设计了多个扩展点，方便进行功能定制和扩展：

| 扩展点 | 说明 | 文件位置 |
|--------|------|---------|
| AI 提示词模板 | 自定义分析、问题生成逻辑 | `app/src/main/resources/prompts/` |
| 评分维度 | 添加新的评分维度 | `ResumeAnalysisEntity`, `InterviewAnswerEntity` |
| 问题类型 | 扩展面试问题类型 | `QuestionType` 枚举 |
| 异步任务 | 添加新的异步处理任务 | 继承 `AbstractStreamConsumer` |
| 文档解析 | 支持新的文档格式 | `FileParserService` |
| 语音服务 | 集成其他语音服务商 | `AsrService`, `TtsService` |

### 3.2 创建新类型的问题生成器

```java
@Component
public class SystemDesignQuestionGenerator {

    private static final String PROMPT_TEMPLATE = """
        你是一个系统设计面试官，请根据候选人的简历生成一道系统设计题目。

        要求：
        1. 题目要与候选人的项目经验相关
        2. 难度适合中高级工程师
        3. 关注高可用、高并发、扩展性

        简历内容：
        {resumeText}

        请返回 JSON 格式：
        {
          "question": "系统设计题目",
          "requirements": ["要求1", "要求2"],
          "evaluationPoints": ["评估点1", "评估点2"]
        }
        """;

    public InterviewQuestionDTO generate(String resumeText) {
        String prompt = PROMPT_TEMPLATE.replace("{resumeText}", resumeText);

        BeanOutputConverter<SystemDesignQuestion> converter =
            new BeanOutputConverter<>(SystemDesignQuestion.class);

        SystemDesignQuestion result = structuredOutputInvoker.invoke(
            chatClient, prompt, converter, ErrorCode.AI_SERVICE_ERROR
        );

        return InterviewQuestionDTO.create(
            0,
            result.question(),
            QuestionType.SYSTEM_DESIGN,
            "系统设计"
        );
    }

    record SystemDesignQuestion(
        String question,
        List<String> requirements,
        List<String> evaluationPoints
    ) {}
}
```

3. 创建消费者：

```java
@Component
@RequiredArgsConstructor
public class SimilarityStreamConsumer extends AbstractStreamConsumer {

    private final ResumeSimilarityService similarityService;

    @Override
    protected String getStreamKey() {
        return RESUME_SIMILARITY_STREAM_KEY;
    }

    @Override
    protected String getGroupName() {
        return RESUME_SIMILARITY_GROUP_NAME;
    }

    @Override
    protected void processMessage(Map<String, String> message) {
        Long resumeId = Long.parseLong(message.get("resumeId"));

        try {
            similarityService.calculateSimilarity(resumeId);
            acknowledgeMessage(message);
        } catch (Exception e) {
            log.error("相似度计算失败: resumeId={}", resumeId, e);
            handleFailure(message, e);
        }
    }
}
```

## 5. 支持新的文档格式

### 5.1 扩展文档解析器

```java
public interface FileParser {
    boolean supports(String contentType);
    String parse(InputStream input) throws IOException;
}

@Component
public class MarkdownFileParser implements FileParser {

    @Override
    public boolean supports(String contentType) {
        return "text/markdown".equals(contentType) ||
               "text/x-markdown".equals(contentType);
    }

    @Override
    public String parse(InputStream input) throws IOException {
        // Markdown 通常不需要特殊解析，直接读取文本
        return new String(input.readAllBytes(), StandardCharsets.UTF_8);
    }
}
```

### 5.2 注册解析器

```java
@Configuration
public class FileParserConfig {

    @Bean
    public List<FileParser> fileParsers(
        PdfFileParser pdfParser,
        WordFileParser wordParser,
        MarkdownFileParser markdownParser
    ) {
        return List.of(pdfParser, wordParser, markdownParser);
    }
}
```

### 5.3 使用解析器

```java
@Service
@RequiredArgsConstructor
public class FileParserService {

    private final List<FileParser> parsers;

    public String parseFile(String contentType, InputStream input) {
        return parsers.stream()
            .filter(parser -> parser.supports(contentType))
            .findFirst()
            .orElseThrow(() -> new BusinessException(ErrorCode.RESUME_FILE_TYPE_NOT_SUPPORTED))
            .parse(input);
    }
}
```

## 9. 插件化扩展

### 9.1 定义插件接口

```java
public interface InterviewPlugin {
    String getName();
    String getVersion();
    void onQuestionGenerated(InterviewQuestionDTO question);
    void onAnswerSubmitted(InterviewAnswerDTO answer);
    void onEvaluationCompleted(InterviewReportDTO report);
}
```

### 9.2 插件管理器

```java
@Component
public class InterviewPluginManager {

    private final List<InterviewPlugin> plugins;

    @Autowired
    public InterviewPluginManager(List<InterviewPlugin> plugins) {
        this.plugins = plugins;
        log.info("加载了 {} 个插件", plugins.size());
    }

    public void notifyQuestionGenerated(InterviewQuestionDTO question) {
        plugins.forEach(p -> {
            try {
                p.onQuestionGenerated(question);
            } catch (Exception e) {
                log.error("插件 {} 执行失败", p.getName(), e);
            }
        });
    }
}
```

### 9.3 实现示例插件
