# 项目概述

## 1. 项目简介

AI-Interview 是一个基于人工智能的智能面试辅导平台，旨在帮助求职者提升面试能力。系统通过 AI 分析简历、生成个性化面试问题、评估面试答案，为用户提供全方位的面试辅导。

### 1.1 核心价值

- **简历智能分析** - AI 自动分析简历内容，提供多维度评分和优化建议
- **个性化面试** - 根据简历内容生成针对性面试问题
- **智能评估** - AI 自动评估面试答案，提供详细反馈
- **知识库问答** - 基于企业知识库的智能问答助手

### 1.2 目标用户

- 准备求职面试的技术人员
- 需要批量筛选候选人的 HR
- 需要面试辅导培训的企业

### 3.2 异步处理架构

系统采用 Redis Stream 实现异步任务处理，提升用户体验：

```
┌─────────────┐     上传      ┌─────────────┐     发送任务      ┌──────────────┐
│   用户请求   │ ──────────> │  Controller │ ───────────────> │ Redis Stream │
└─────────────┘               └─────────────┘                  └──────────────┘
                                                                    │
     ┌──────────────────────────────────────────────────────────────┘
     │
     ▼
┌─────────────┐     消费任务     ┌─────────────┐     调用 AI      ┌─────────────┐
│   用户响应   │ <────────────── │  Consumer   │ ──────────────> │  AI 服务    │
└─────────────┘                 └─────────────┘                  └─────────────┘
  (立即返回)                      (后台处理)                      (异步执行)
```

**异步任务类型**:

| 任务 | Stream Key | 消费者组 | 处理器 |
|------|-----------|---------|--------|
| 简历分析 | `resume:analyze:stream` | `analyze-group` | `AnalyzeStreamConsumer` |
| 面试评估 | `interview:evaluate:stream` | `evaluate-group` | `EvaluateStreamConsumer` |
| 向量化 | `knowledgebase:vectorize:stream` | `vectorize-group` | `VectorizeStreamConsumer` |

## 4. 技术亮点

### 4.1 Java 21 虚拟线程

```java
// 虚拟线程配置
@Configuration
public class ThreadConfig {
    @Bean
    public Executor taskExecutor() {
        return Executors.newVirtualThreadPerTaskExecutor();
    }
}
```

**优势**:
- 轻量级并发，可创建百万级虚拟线程
- 适用于 I/O 密集型场景 (AI 调用、文件操作)
- 显著提升系统吞吐量

### 4.2 pgvector 向量搜索

```sql
-- 向量相似度搜索
SELECT * FROM knowledge_chunks
ORDER BY embedding <=> '[0.1, 0.2, ...]'
LIMIT 10;
```

**应用场景**:
- 知识库文档检索
- 相似简历推荐
- 语义搜索

// 任务生产者
